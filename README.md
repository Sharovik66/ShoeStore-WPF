## Технологический стек
* **Платформа:** .NET 8.0 (WPF)
* **Язык программирования:** C# (ООП, Архитектурный паттерн MVP)
* **База данных:** Реляционная СУБД MySQL 8.0
* **Компоненты сборки:** `MySql.Data`, `System.Windows.Media`

## Структура репозитория
* `Практика (2).zip` — Полный исходный код приложения (Слои Models, Views, Presenters) и графические ресурсы.
* `practica.sql` — Полный скрипт базы данных (структура 8 таблиц, каскадные связи и 20 позиций товаров).
* `ERD-diagramma (practica).pdf` — Официальная ER-диаграмма сущностей и связей базы данных СУБД.
* `отчет 1.docx` — Протокол комплексной отладки и приемо-сдаточного функционального тестирования ИС.

## Ключевые особенности и функции безопасности
1. **Паттерн MVP (Model-View-Presenter):** Бизнес-логика полностью отделена от графического представления интерфейса.
2. **Аппаратная защита Mutex:** Реализован запрет на запуск второго экземпляра приложения в ОС Windows.
3. **Безопасность СУБД (RESTRICT):** База данных аппаратно блокирует удаление товаров, присутствующих в активных заказах клиентов.
4. **Валидация данных:** Использование `CultureInfo.InvariantCulture` защищает систему от переполнения числовых буферов MySQL независимо от локализации Windows.
5. **Оптимизация ресурсов:** Автоматическое сжатие прикрепляемых администратором картинок до экзаменационных 300х200 пикселей.
